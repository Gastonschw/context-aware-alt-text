Context-Aware Alt Text Support

Version 1.1.0

A Chrome extension that detects images missing alt text and generates context-aware descriptions using AI to improve web accessibility.

How It Works

  1. Detect

    Scans every image on the page. Images missing alt text are outlined in red, decorative images in orange, and images with alt text in green.

  2. Generate

    Sends the image along with surrounding page context to the TAMU AI vision API, which returns a concise, accurate description.

  3. Review

    Each suggestion is shown in an editable panel. You can accept, reject, edit, or mark images as decorative — individually or in bulk.

  4. Persist

    Accepted alt texts are saved locally and survive page reloads, so you don't lose your work.

Keyboard Shortcuts

These shortcuts work inside the extension popup.

ShortcutAction
Alt + GGenerate alt text for the selected image
Alt + AAccept the current suggestion
Alt + RReject the current suggestion
Alt + Shift + AAccept all suggestions
Alt + Shift + RReject all suggestions

Getting Started

  1. Open Settings from the extension popup and enter your TAMU API server and API key.
  2. Navigate to any webpage and click the extension icon to see the accessibility score.
  3. Click Generate All Alt Text or generate descriptions for individual images.
  4. Review, edit, and accept the suggestions. They will be saved automatically.

Color Guide

Red outline — Image is missing alt text
Orange outline — Image is decorative
Green outline — Image has alt text
